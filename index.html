About

This is a demonstration app and under active development.

This is a concept app that allows users to enter a parcel ID (pin number) and get a listing of the zoning district(s) that intersect the parcel. Testing is still required to ensure the analysis works properly.

It should also be noted this app is only as good as the underlying zoning district data maintained by Cumberland County (PA). Below are some sample PIN numbers to test the app. Another option would be to allow users to search by PIN or address. The municipality for the corresponding address could be shown in the hint bar for results.

Sample PIN Numbers:

  • Parcel with single zoning district: 40-31-2187-062
  • Parcel with multiple zoning districts: 22-12-0350-052
  • Parcel number that does not exist: 52-01-123-004
